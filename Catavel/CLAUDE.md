## Apps Script
- **Current version**: v12.1
- **Key functions**:
  - `mainPipeline()` — full run every 6 hours (time-driven trigger)
  - `onEdit(e)` — simple trigger, fires on Vouchers tab edit → `exportVouchersJSON()`
  - `exportVouchersJSON()` — push vouchers.json to GitHub
  - `postNewVouchersToTelegram()` — scheduled 3x/day (09:00, 13:00, 19:00 WIB)
  - `pushVouchersManual()` — force push vouchers.json
  - `testConnection()` — test all APIs
  - `setupVouchersSheet()` — create Vouchers tab with headers

## Vouchers Tab Schema (13 cols, 0-indexed)
```
0:OTA | 1:Promo Code | 2:Description | 3:Type | 4:Discount
5:Min Spend | 6:Max Discount | 7:Applicable For | 8:Valid Until
9:Market | 10:Affiliate Link | 11:T&Cs | 12:Status
```
- **Status values**: `active`, `expired`
- **Market values**: `SGD`, `MYR`, `HKD`, `PHP`, `ALL`
- **Note**: Google Sheets returns percentage cells as decimals (8% → 0.08). Script handles normalization.
